# Cómo Subir Cambios a GitHub desde la Web

## Si ya tienes un repositorio en GitHub:

### Paso 1: Ir a tu repositorio en GitHub
1. Abre tu navegador y ve a: `https://github.com/tu-usuario/tu-repositorio`
2. O busca tu repositorio en GitHub

### Paso 2: Subir archivos individuales
1. Haz clic en el archivo que quieres actualizar (por ejemplo: `src/App.jsx`)
2. Haz clic en el ícono de lápiz (✏️) en la esquina superior derecha para editar
3. Copia y pega el contenido completo del archivo
4. Haz clic en "Commit changes" al final de la página
5. Escribe un mensaje de commit: "Optimización de procesamiento de archivos grandes y dashboard"
6. Haz clic en "Commit changes"

### Paso 3: Crear archivo nuevo
1. En tu repositorio, haz clic en "Add file" → "Create new file"
2. Escribe el nombre del archivo: `ALERTAS_PRESUPUESTO.md`
3. Copia y pega el contenido del archivo
4. Haz clic en "Commit new file"

## Si NO tienes un repositorio aún:

### Paso 1: Crear nuevo repositorio
1. Ve a https://github.com/new
2. Nombre del repositorio: `sistema-sureste` (o el nombre que prefieras)
3. Elige si será público o privado
4. **NO marques** "Initialize this repository with a README"
5. Haz clic en "Create repository"

### Paso 2: Subir archivos
1. GitHub te mostrará instrucciones para subir archivos
2. Haz clic en "uploading an existing file"
3. Arrastra y suelta TODOS los archivos de tu carpeta `sistema-sureste`:
   - `src/App.jsx`
   - `ALERTAS_PRESUPUESTO.md`
   - `README.md`
   - Y todos los demás archivos
4. Escribe un mensaje de commit: "Subida inicial del sistema Sureste con dashboard y optimizaciones"
5. Haz clic en "Commit changes"

## Archivos importantes a subir:

✅ **src/App.jsx** - Contiene el dashboard y optimizaciones
✅ **ALERTAS_PRESUPUESTO.md** - Guía de alertas de presupuesto
✅ **README.md** - Documentación del proyecto
✅ Todos los demás archivos del proyecto

## Nota:
Si GitHub Desktop no funciona, esta es la forma más directa de subir tus cambios.
